## Overview

**Cashflow** is a backend project built with **.NET 9** as a **Web API** designed for studying software architecture, backend development, persistence, and automated testing.  
The project is currently under active development and serves as a practical environment for experimenting with real-world backend patterns and tools.

All development is being done with an educational purpose, aiming to practice clean code, layered design, and integration with relational databases.

## Purpose

This project was created **exclusively for learning and experimentation**, with the following goals:

- Strengthen backend development skills with .NET 9  
- Practice clean architecture and structured API design  
- Understand relational persistence using PostgreSQL  
- Use Docker for infrastructure provisioning  
- Implement and maintain unit and integration tests  
- Build a foundation for more complex backend projects
